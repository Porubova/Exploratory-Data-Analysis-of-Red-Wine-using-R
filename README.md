# Exploratory-Data-Analysis-or-Red-Wine-using-R
Application of EDA techniques to explore relationships between different variable using R.
## Dataset
This tidy data set contains 1,599 red wines with 11 variables on the chemical properties of the wine. At least 3 wine experts rated the quality of each wine, providing a rating between 0 (very bad) and 10 (very excellent). Read more about dataset.[a link](https://s3.amazonaws.com/udacity-hosted-downloads/ud651/wineQualityInfo.txt)
